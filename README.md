# Modern React Portfolio

A premium, high-performance portfolio website built with React, Vite, Framer Motion, and Tailwind CSS.

## Features
- **Modern Aesthetics**: Dark mode with glassmorphism and radiant gradients.
- **Dynamic Animations**: Smooth scroll reveals and hover effects using Framer Motion.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **Project Showcase**: Beautifully designed project cards with image support.
- **Skill Tags**: Interactive skill badges.

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation
1. Navigate to the project directory:
   ```bash
   cd portfolio-website
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Customization
- **Content**: Update projects, skills, and personal info in `src/App.jsx`.
- **Styling**: Modify theme colors and global styles in `src/index.css`.
- **Assets**: Add your own project images to the `public/` folder.

## Technologies Used
- **React**: UI Framework
- **Vite**: Build Tool
- **Framer Motion**: Animations
- **Lucide React**: Icons
- **Tailwind CSS**: Utility-first styling
